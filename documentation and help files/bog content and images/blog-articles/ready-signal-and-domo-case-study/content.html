Ready Signal and Domo – Case Study

Improve Machine Learning Model Accuracy in Domo with Automated Feature Engineering and Control Data from Ready Signal

The video demonstrates an understanding of how control data is used by those building predictive models. Many analysts and data scientists dismiss the use given the challenges of sourcing and preparing the data for incorporation into models. Ready Signal has created software to address these challenges.  Through this example, we demonstrate the ease with which to obtain and curate data, build them into a predictive model and display the models using Domo.

The goals we hope to share are threefold:

  1. Demonstrate the accuracy improvement in predictive models when using control data
  2. Demonstrate how challenging, and time consuming to source, normalize and make assessable Control data
  3. Demonstrate how Ready Signal delivers fully curated control data directly into Domo through a Domo Data Connector
